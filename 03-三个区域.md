# Git的三个区域
## Git使用时：
**工作区：** 实际==开发==时操作的文件夹
**暂存区：** 保存之前的==准备区域==（暂存改动过的文件）（.git/index)
**版本库：** 提交并==保存==暂存区中的内容，产生一个版本快照（.git/objects）
## 命令：
- 暂存指定文件
`git add 文件名`

- 暂存所有改动的文件
`git add .`

- 提交保存，产生版本快照
`git commit -m "注释说明"`

- 查看暂存区的文件：
`git ls-files`
